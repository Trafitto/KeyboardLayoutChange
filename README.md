# Keyboard Layout Change - Extension for Ulauncher

As the name says, a simple extension for Ulauncher that allows me to switch the keyboard layout quickly

Currently it only supports 2 layouts (the ones I use)

#### Layout supported

    - US Qwerty
    - US ALT INT
    - US Colemak-DH
    - IT
    - ES
    - FR
    - PL
    - PT-BR
    - DE
